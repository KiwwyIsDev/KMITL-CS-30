ตัวเลขที่มากที่สุดบนสามเหลี่ยมลูกบาศก์สุดมหัศจรรย์
Problem Setter : Momo-san the impersonator
จากลูกบาศก์จำนวนมากที่อาจารย์โมซื้อมา อาจารย์โมต้องการสร้างสามเหลี่ยมจากลูกบาศก์ตัวเลขที่มีลักษณะเช่นนี้

สังเกตว่าใน N = 3 ตัวเลข 2 คือผลบวกระหว่างตัวบนซ้าย (ตัวเลข 1) และบนขวา (ตัวเลข 1) ที่เยื้องกับตัวเลข 2 หากกำหนดให้อาจารย์โมต้องการสร้างลูกบาศก์ทรงสามเหลี่ยมขนาด N = M แล้วอาจารย์โมอยากรู้ว่าตัวเลขที่มีค่ามากที่สุดในสามเหลี่ยมเป็นเท่าใด

รูปแบบข้อมูลนำเข้า (Input)
Input มีทั้งหมด 1 บรรทัด เป็นจำนวนเต็มบวก M โดย 1≤M≤50000

รูปแบบข้อมูลการแสดงผล (Output)
Output ที่ต้องการมีทั้งหมด 1 บรรทัดเป็นจำนวนเต็มแทนตัวเลขในสามเหลี่ยมที่มีค่ามากที่สุด โดยต้อง mod 109+7

ตัวอย่าง (Test Case)
Input	Output
1	1
2	1
10	126
